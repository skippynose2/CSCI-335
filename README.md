# Determining Stock Movement 1, 2, and 3 days After Earnings Calls
by Akhil Devarapalli, Saadhan Pittala, and Neav Ziv


## Abstract
The goal of this project was to try and predict stock outcomes after observing trends, reports, and other forms of company data 
publicly available online. Many different models and algorithms were built on the data, including SVM, Logistical Regression, and
Decision Trees, resulting in the discovery that different models work more efficiently depending on the amount of days of movement
being evaluated. Ultimately, the models that worked best for the 1, 2, and 3 day movements were logistic regression, a tie between 
logistic regression and SVM, and SVM respectively. Conclusively, the data and results show that while they do not work as standalone
buy decisions, these models are definitely powerful tools that can greatly assist in trade analytics.

## How to Run the Program
Navigate to the code directory and run the file with py file-name.py. The file is named after the model it is.
